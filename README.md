This repository contains supplementary files related to 'Deep learning for medical ultrasound image segmentation: A systematic review of the current research' by Rainio et al.

The file us_dois.csv contains the original DOI codes of the initial PubMed search results in order, and details of the screening process.
The file us_dois.xlsx contains the same information reorganized and with relevant column titles.
The file us_final.xlsx includes the included articles and the details collected from them.
The file usr.R contains codes used to compute statistics, run statistical tests, and produce figures.
